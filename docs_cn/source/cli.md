# 命令行


## 启动服务

```bash
advisor_admin server start
```

## 停止服务

```bash
advisor_admin server stop
```

## 提交任务

```bash
advisor run -f ./advisor_client/examples/python_function/config.json
```

## 列举Study

```bash
advisor study list
```

## 查看Study

```bash
advisor study describe -s demo
```

## 列举Trial

```bash
advisor trials list
```
