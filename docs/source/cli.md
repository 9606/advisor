# Command Line Interface


## Start Server

```bash
advisor_admin server start
```

## Stop Server

```bash
advisor_admin server stop
```

## Submit Job

```bash
advisor run -f ./advisor_client/examples/python_function/config.json
```

## List Study

```bash
advisor study list
```

## Describe Study

```bash
advisor study describe -s demo
```

## List Trials

```bash
advisor trials list
```
